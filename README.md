# Koperasi Charta Cempaka — Website

Tugas pemrograman web: Website koperasi simpan pinjam dengan studi kasus pegadaian.

## Struktur Folder

```
koperasi-charta-cempaka/
├── index.html          → Landing Page (Halaman Utama)
├── pendaftaran.html    → Halaman Form Pendaftaran Anggota
└── README.md           → Dokumentasi ini
```

## Halaman

### 1. `index.html` — Landing Page
Berisi:
- Navbar sticky dengan navigasi dan CTA
- Hero section dengan simulasi pinjaman
- Profil koperasi (Visi, Misi, Nilai Inti)
- Section layanan (Simpan Pinjam, Gadai, Tabungan)
- Syarat & Ketentuan keanggotaan
- CTA section menuju pendaftaran
- Footer

### 2. `pendaftaran.html` — Form Pendaftaran
Berisi:
- Layout 2 kolom: panel info kiri + form kanan
- Step indicator (3 langkah)
- Progress bar
- Form: Data Pribadi, Kontak & Domisili, Pekerjaan
- Radio card pekerjaan interaktif
- Validasi sederhana sebelum submit
